# CUBE
it's literally just a cube. but it rotates.

all the math was done based on some wikipedia pages for
the rotation matrix and stuff like that.
all the rendering was done with ANSI codes and printf.

# how to run
```sh
gcc -o cube cube.c
./cube
```
